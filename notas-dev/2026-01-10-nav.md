Estilos CSS a la Navegación
Resumen del concepto
Estilizar la navegación consiste en aplicar reglas CSS a elementos <nav> y sus enlaces para crear barras de menú visualmente atractivas y funcionales. Incluye fondo, padding, hover effects, centrado y clases utilitarias como .nav-background para extender el fondo a todo el ancho mientras el contenido permanece centrado. En desarrollo web moderno es clave para mejorar UX, accesibilidad y consistencia visual, ya que una navegación clara y responsiva es el primer punto de interacción del usuario. Usar clases específicas y atajos como margin: 0 auto optimiza el código y el mantenimiento.
Lo que más me costó entender
Me costó comprender por qué el fondo del <nav> no se extendía al ancho completo y cómo centrar el contenido sin afectar el fondo. También me confundió el uso de margin: auto (no funcionaba al principio) y cómo combinar clases utilitarias sin romper la semántica.
Cómo lo resolví
Inspeccioné en DevTools para ver márgenes y anchos reales. Experimenté con width y margin: 0 auto hasta ver que necesitaba un contenedor con ancho fijo (ej: 120rem). Probé clases utilitarias (.nav-background) para separar fondo y contenido. Un tip de Traversy Media sobre "container + full-width background" me ayudó a resolverlo definitivamente.
Antes (fondo no se extiende):
CSSnav { background: var(--color-secundario); }
Después (fondo completo + contenido centrado):
CSS.nav-background { background: var(--color-secundario); }
.contenedor { width: 120rem; margin: 0 auto; }
Ejemplo práctico aplicado en mi proyecto
En Freelancer creé una clase utilitaria para fondo completo y contenedor centrado.
HTML<div class="nav-background">
    <nav class="navegacion-principal contenedor">
        <a href="#">Inicio</a>
        <a href="#">Sobre mí</a>
        <!-- ... -->
    </nav>
</div>
CSS.nav-background {
    background: var(--color-secundario); /* Fondo azul completo */
}

.contenedor {
    width: 120rem;          /* 1200px */
    margin: 0 auto;         /* Centrado horizontal */
    padding: 1rem 0;
}

Conclusión / Aprendizaje clave
Aprendí que separar fondo completo (clase utilitaria) del contenido centrado es una práctica limpia y escalable. Usar margin: 0 auto y contenedores con ancho fijo me da control total sobre layouts. Este enfoque me hace más eficiente al escribir CSS mantenible y me prepara para diseños responsivos complejos.
